# AxiOwl Mailbox

AxiOwl Mailbox is a built-in local endpoint and GUI for receiving messages, inspecting provider results, and running bounded provider test plans.

## Capabilities

| Operation | Status |
|---|---|
| Discovery | built in |
| Receive/send target | supported |
| Create | unsupported singleton |
| Rename | unsupported reserved name |

The mailbox is not an external provider and cannot be duplicated through provider create. It is installed with the core runtime.
